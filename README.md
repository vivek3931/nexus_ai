# Nexus AI

Nexus AI is a full-stack web application built with React, plain CSS, Node.js, and MongoDB. The core idea is to use Google to search for images related to user input, while leveraging AI to generate intelligent text outputs, providing a seamless and interactive user experience.

## Features

- Modern React frontend
- Custom styling with plain CSS
- RESTful API server with Node.js
- MongoDB for data storage
- Google-powered image search based on user input
- AI-generated text responses
- Scalable and modular architecture

## Tech Stack

- **Frontend:** React, CSS
- **Backend:** Node.js, Express
- **Database:** MongoDB

## Getting Started

### Prerequisites

- Node.js & npm
- MongoDB

### Installation

1. Clone the repository:
    ```bash
    git clone https://github.com/yourusername/nexus-ai.git
    cd nexus-ai
    ```

2. Install dependencies for frontend and backend:
    ```bash
    cd frontend
    npm install
    cd ../backend
    npm install
    ```

3. Configure environment variables as needed.

4. Start the backend server:
    ```bash
    npm start
    ```

5. Start the frontend:
    ```bash
    cd ../frontend
    npm start
    ```

## Folder Structure

```
/frontend   # React app
/backend    # Node.js server
```
